## 🔟 확장 아이디어

### 1. 결제 연동

#### 계좌 자동 동기화
- **은행 API 연동**
  - 오픈뱅킹 API를 통한 계좌 잔액 실시간 조회
  - 거래 내역 자동 동기화
  - 주요 은행: KB, 신한, 우리, 하나, NH농협

#### 카드 결제 연동
- **카드사 API 연동**
  - 카드 결제 내역 자동 수집
  - 카드 할부 정보 관리
  - 포인트/마일리지 적립 현황

**기술 스택**
- Open Banking API
- OAuth 2.0 인증
- 배치 작업으로 주기적 동기화

### 2. OCR 영수증 인식

#### 영수증 이미지 자동 인식
- **기능**
  - 영수증 사진 촬영
  - OCR을 통한 거래 정보 자동 추출
  - 카테고리 자동 분류
  - 금액, 날짜, 가맹점명 자동 입력

#### OCR 서비스 옵션
- **Google Cloud Vision API**
- **AWS Textract**
- **Naver Clova OCR**
- **Kakao OCR**

**구현 방안**
```
1. 사용자가 영수증 사진 업로드
2. S3에 이미지 저장
3. Lambda/EC2에서 OCR API 호출
4. 추출된 데이터를 거래 내역으로 자동 생성
5. 사용자 확인 후 저장
```
